The notebook is the single place that runs and narrates the whole project so far: preprocessing (W1–2), U-Net segmentation (W3–4), the Attention U-Net (W5) and tumour-subtype classification (W6).

Read first — honest data status. No real CATCH slides have been processed yet; every figure in the notebook is computed on synthetic slides while the (license-gated, ~hundreds-of-GB) TCIA download is arranged. The synthetic "tumours" and "subtypes" are drawn shapes/colours, so the numbers validate the code, not the histology. The pipeline is now real-data-ready — see "Running on real CATCH" below.

Step by stepOpening the notebook

Jupyter often opens at your home folder (e.g. C:\Users\Admin). You must navigate into the project so the imports resolve.

Start Jupyter from the project folder so it opens in the right place: 
cd E:\Project\canine-melanoma-catch
.venv\Scripts\Activate.ps1
jupyter lab notebooks\CATCH_Melanoma_Project.ipynb
Or, if Jupyter is already open at your home folder, click through to E: → Project → canine-melanoma-catch → notebooks and open the file.
Run all cells — menu Run → Run All Cells. The notebook reads the saved results in outputs/ and renders figures; it does not retrain, so it runs in seconds.
If a path error appears, the first code cell auto-detects the project root (the folder containing config/config.yaml) and chdirs into it — just make sure you opened the notebook from inside the project, not a copied-out file.

How to present this honestly. The Attention U-Net clearly leads (Dice 0.965). The ResNet-34 encoder's low score is under-training noise from a 2-epoch CPU run with a high BCE pos_weight (~9.6) — not a real ranking (note its pixel-acc 0.53 = over-prediction). These validate the pipeline on easy synthetic masks; absolute values and the encoder comparison will change with full-length training on real CATCH annotations.

Classification — ResNet-50 (synthetic demo, 3 classes)

Accuracy / macro-F1 / AUC = 1.000. Present this as: "the training/eval loop is correct" — not subtype discrimination. The demo's three classes are colour-separable by construction, so a perfect score is expected. On real CATCH this becomes a 7-class, class-imbalanced problem handled by class-weighted cross-entropy and the slide-level stratified split.

For the real runRunning on real CATCH

Once the slides are downloaded (TCIA DOI 10.7937/TCIA.2M93-FX66; needs OpenSlide binaries), use the recommended subtype-folder layout and run without --demo:

data/raw/
  Melanoma/*.svs   Mast cell tumor/*.svs   ...   Plasmacytoma/*.svs
  annotations/CATCH.json     # MS-COCO polygons (tumour masks)

python -m src.data_acquisition.download_catch              # instructions + verify
python scripts/prepare_segmentation_data.py  --input data/raw   # WSI tiles + masks
python scripts/train_unet.py --arch attention
python scripts/prepare_classification_data.py --input data/raw  # tumour tiles + subtype
python scripts/train_classifier.py --arch resnet50 --epochs 20
