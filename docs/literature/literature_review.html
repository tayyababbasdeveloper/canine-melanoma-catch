!Identified gap (restated)

Research gap this project fills

Strong canine–human melanoma parallels + a curated dataset (CATCH) exist, but few studies apply modern segmentation/classification deep learning specifically to canine melanoma. This project fills that gap and tests translational relevance to human non-UV melanoma.
