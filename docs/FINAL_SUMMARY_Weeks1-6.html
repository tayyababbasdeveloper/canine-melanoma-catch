Supervisor's notes — addressedAnswers to this week's questions

Q. Plan the classification section — what is the segmentation output?
The segmentation outputs a per-pixel binary tumour mask (tumour vs background) for each patch. It is used to (a) locate the tumour and (b) select the tumour patches that are then fed to the classifier. Full plan in docs/CLASSIFICATION_PLAN.md and notebook §0.
Q. What are you going to classify? How many classes?
The tumour subtype. CATCH has 7 subtypes (melanoma, mast cell, squamous cell carcinoma, peripheral nerve sheath tumour, trichoblastoma, histiocytoma, plasmacytoma). The classifier supports 7-class or melanoma-vs-non-melanoma; the demo uses 3 representative classes.
Q. Are there enough of each class? What about balancing?
CATCH is imbalanced; the pipeline measures and reports per-class counts every run. Balancing is handled with class-weighted cross-entropy, augmentation, and a stratified slide-level split (implemented in src/training/classifier_trainer.py).
Q. Balancing — before or after the segmentation?
Two different imbalances, two stages: tumour-vs-background (~9 %) is handled inside segmentation (BCE pos_weight + Dice); subtype class imbalance is handled after segmentation, in classification — because the subtype label only matters once tumour tissue is isolated.
Q. Are you using the full dataset? Only 8 images?
The 616 figure is extracted patches, not slides (one slide → many patches at 5×/10×/20×). Development is validated on synthetic slides while the full 750-slide CATCH download from TCIA completes; the same code runs unchanged on the full dataset via --input data/raw (notebook §7).
Q. Attention U-Net variant.
Implemented (Oktay 2018 attention gates on the skip connections) in src/models/attention_unet.py, trained, and compared three-way against the baseline and ResNet-34 U-Nets. ✓
Q. Set up your work on a Jupyter notebook.
Done. notebooks/CATCH_Melanoma_Project.ipynb runs and explains the whole project (W1–6), including the classification plan, live preprocessing demo, the model comparisons, and the full-dataset instructions. ✓
